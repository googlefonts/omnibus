A New Era for Editorial Typography

Engineered for the press and editorial sector, Press Series is a collection of typeface families that spans the full spectrum of publishing. Divided into sans-serif and serif, it seamlessly bridges extensive and immersive reading. The sans designs excel in headlines and navigation, while the serif counterparts ensure maximum readability for dense text, delivering typographic excellence from traditional books to digital newspapers.

    About
    

      

        
Press Series is an open-source collection features six variable fonts optimized for continuous reading, delivering a flawless aesthetic experience both on high-resolution screens and in physical print. Each family is inspired by the traditions of print type and drawn with generous proportions for immersive, long-form reading.

        
Every family now ships as an open-source variable font: a single file carries the whole design space — continuous weight, and, where the design calls for it, width and italics. Prefer fixed styles? Each is also available as a set of static fonts.
